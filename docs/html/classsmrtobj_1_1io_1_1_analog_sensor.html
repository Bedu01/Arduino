Detailed Description

The AnalogSensor class model a generic analog sensor. It measures the voltage on a analog input of the Arduino board. The input voltage (Vin) is given by the following formula:

\[Vin= Nadc*(Vref/2^n)\]

Where: Nadc is the output value of the ADC;
Vref is the reference voltage of the ADC;
n is the number of bits used for the conversion.

Vref can be the changed the Arduino analogReference function and smrtobj::AnalogSensor::setReference (or smrtobj::AnalogSensor::setReferenceDefault) functions. This change applies to all analog inputs.

//analogReference(DEFAULT);
smrtobj::AnalogSensor::setReferenceDefault(smrtobj::AnalogSensor::V5_0);
analogReference(EXTERNAL);
smrtobj::AnalogSensor::setReference(2.5);
